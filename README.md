😉현주 - 서로 도우면서 각자 맡은 역할을 열심히 하여서 협업의 재미를 알 수 있었습니다. 프로젝트 진행하며 막힌 부분이 생기고 포기하고 싶을 때 다 함께 해결하고자 하는 마음을 모으니 힘이 되었습니다. 기능 구현을 하면서 배운 내용을 복습할 수 있었습니다.
